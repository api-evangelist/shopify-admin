Inventory transfer webhooks now include origin and destination location IDs as Global IDs, eliminating the need for additional API calls to identify transfer sources and destinations. Documentation for inventoryTransferSetItems and inventoryTransferRemoveItems mutations has been clarified to better explain their behavior, and error messages have been improved with more descriptive explanations.
