Shop User is Shopify’s cross-merchant buyer identity. Partners who use metafields on Shop Users can now read those metafields during checkout using Shopify Functions. To learn more, see the Shop User metafields guide in the Shopify developer documentation: https://shopify.dev/docs/api/shop/guides/use-cases/metafields
